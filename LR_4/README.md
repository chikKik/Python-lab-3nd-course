
Для запуска сервера uvicorn использовалась команда:</br>
``` python -m uvicorn LR_4:app ```

Глоссарий поддерживает следующие операции:</br>

- Получение списка всех терминов.</br>
- Получение термина по его ID.</br>
- Получение информации о конкретном термине по ключевому слову.</br>
- Добавление нового термина с описанием.</br>
- Обновление существующего термина.</br>
- Удаление термина из глоссария.</br>
